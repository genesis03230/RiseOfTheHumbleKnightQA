## TC-01: Select and Execute a Combat Action (Happy Path)

**Test Type:** Manual  
**Test Level:** Gameplay / Functional  
**Priority:** High  

### Preconditions
- Game launched successfully
- Player has reached a turn-based combat encounter
- Combat UI is visible and interactive

### Steps
1. Start a combat encounter
2. Wait for the player's turn
3. Select an offensive action from the combat UI
4. Confirm the selected action
5. Observe the action execution

### Expected Result
- The selected action is executed correctly
- Combat flow advances to the next turn
- UI updates correctly to reflect the new combat state
- No unintended input locks occur
